# OncoAtlas 数据目录

本目录将原结构化数据集转换为 Markdown 表格，便于在 GitHub 中直接阅读、搜索和引用。

| 数据集 | 内容 |
| --- | --- |
| [药物数据库](drugs.md) | 代表性肿瘤药物、靶点、适应症、关键试验和机制摘要 |
| [管线数据库](pipelines.md) | 临床管线、治疗模式、阶段、试验编号和预计读出 |
| [公司数据库](companies.md) | 公司、国家/地区、研发重点、旗舰产品和平台 |
| [FDA批准记录](fda_approvals.md) | FDA 肿瘤学批准、审批路径、适应症和关键试验 |
| [术语缩写表](terms.md) | 肿瘤学、药理学、临床试验和监管缩写 |

维护原则：更新正文时同步更新本目录中的表格；快速变化的信息需标注数据截止时间并回到权威来源复核。
