# OncoAtlas — 癌症治疗前沿知识数据库

## 数据口径

- 内容版本：2025.Q2。
- 适用于教育和研究，不构成医疗、监管或投资建议。
- 快速变化的信息（批准状态、价格、交易、临床阶段）应回到 FDA、ClinicalTrials.gov、公司公告和同行评议论文复核。
